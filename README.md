## 🌟 项目简介

**核桃雅集** 是一款专注于文玩核桃收藏与分享的 Web 应用。我们致力于通过艺术化的展示方式，呈现每一对核桃随时间流逝而产生的独特包浆与质感，为玩家提供一个纯粹、雅致的交流空间。

> 每一对核桃都承载着时间的记忆。从青皮到红润，从粗糙到玉化，这不仅是把玩的过程，更是修心的旅程。

## ✨ 核心功能

- 🖼️ **沉浸式画廊**：基于 Masonry 布局的精美藏品牆，多维度展示核桃细节。
- 🔍 **深度详情**：支持多图切换，查看核桃的品种、重量、尺寸及包浆状态。
- 🎨 **分享海报**：一键生成排版精美的 3:4 分享海报（支持二维码），方便社交分享。
- 📍 **分类索引**：快速筛选狮子头、官帽、虎头等经典品种。
- 📤 **藏品上传**：简单易用的上传流程，记录您的收藏故事。

## 🛠️ 技术栈

- **前端框架**: React 18
- **构建工具**: Vite
- **样式处理**: Tailwind CSS (Lucide-react 图标库)
- **动画效果**: CSS Transitions & Tailwind Animate

## 🚀 快速开始

### 前置条件

- 已安装 [Node.js](https://nodejs.org/) (建议 v16.0.0+)

### 安装与运行

1.  **克隆或克隆下载项目**后进入目录：

    ```bash
    cd hetaozhi
    ```

2.  **安装依赖**:

    ```bash
    npm install
    ```

3.  **配置环境变量**:
    在项目根目录创建或编辑 `.env.local` 文件，设置您的 API Key：

    ```env
    GEMINI_API_KEY=your_gemini_api_key_here
    ```

4.  **启动开发服务器**:
    ```bash
    npm run dev
    ```
    启动后访问 `http://localhost:5173` 即可查看。
